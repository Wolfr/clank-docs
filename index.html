Clank

An open source HTML/CSS framework for prototyping native phone/tablet applications.

View demos Download

Free & open source

Clank is free and open source and it will remain so. We welcome improvements and suggestions on Github.

Component based

Pick and choose. Just like known frameworks like Twitter Bootstrap, pick what you need and use it.

Modern SCSS

The Clank code is written using modern CSS techniques. Under the hood it utilizes Sass and Compass to ensure a maintainable codebase.

As automatic as possible

Use generators to quickly create new pages. Get up to speed with Grunt tasks. Automatically create icon fonts from vector files.
